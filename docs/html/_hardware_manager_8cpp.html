Detailed Description

Implements the HardwareManager class which controls all low-level hardware interfaces.

This includes initialization of ADC, encoder, digital buttons, shift registers, MUX selection logic, EEPROM configuration storage, and serial parameter transmission to the Roland JX-8P synthesizer.

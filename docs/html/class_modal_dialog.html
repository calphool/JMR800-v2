Detailed Description

Base class for modal overlays requiring user dismissal or confirmation.

Modal dialogs are interactive UI layers that block interaction with the underlying screen until they are dismissed. Derived classes must implement draw logic, lifecycle hooks, and a mechanism to indicate completion.

Detailed Description

Declares the ScreenManager class responsible for UI screen control on the OLED display.

The ScreenManager handles the active UI screen logic, regularly invokes its loop method, and manages access to the shared Adafruit_SH1106G OLED display used throughout the interface.

This is part of the JMR800-V2 project running on a Teensy 4.1 microcontroller.
